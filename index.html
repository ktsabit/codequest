About

CodeQuest merupakan kompetisi web development yang diselenggarakan oleh BEMP Ilmu Komputer UNJ. Kompetisi ini bertujuan untuk mengasah kemampuan mahasiswa dalam membuat website yang kreatif dan inovatif. CodeQuest 2023 mengusung tema “Menggali Kreativitas dan Inovasi dalam Pembuatan Website untuk Tujuan Pembangunan Berkelanjutan (SDGs).”.

Syarat dan Ketentuan

Syarat Ketentuan
  • Peserta merupakan mahasiswa aktif program studi S1/D4.
  • Peserta mampu menyelesaikan perlombaan sebelum batas waktu.
  • Peserta dapat membuat website sesuai dengan tema SGDs.
  • Peserta dapat mempertanggungjawabkan orisinalitas karyanya.
  • Peserta tidak boleh melakukan plagiarisme karya dari orang lain.

Tools dan Teknologi:

Peserta diperbolehkan menggunakan berbagai alat dan teknologi front-end seperti HTML, CSS, JavaScript, dan framework seperti React, Angular, atau Vue.js. Penggunaan perangkat lunak desain grafis seperti Adobe atau Figma juga diperbolehkan untuk perancangan tampilan. Penggunaan platform atau framework tertentu (seperti WordPress, Drupal, Bootstrap, Tailwind, dll.) tidak diperbolehkan dalam lomba.

Tampilan Situs Web:

Situs web yang dikembangkan harus memiliki desain front-end yang menarik, kreatif, dan responsif agar dapat diakses dengan baik di berbagai perangkat, termasuk perangkat seluler. Kualitas tampilan dan keberlanjutan antarmuka pengguna akan dinilai.

Bahasa Pemrograman:

Peserta dapat menggunakan bahasa pemrograman yang sesuai untuk pengembangan front-end, seperti HTML, CSS, dan JavaScript.

Pengumpulan Hasil:

Peserta dapat mengumpulkan hasil perlombaan dalam bentuk file (.zip). Hasil perlombaan wajib dideploy ke dalam GitHub Page, Netlify, atau sejenisnya.
